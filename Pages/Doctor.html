Doctor Who Desktop Cycler

Small Applescript program that switches your desktop background to match the hour

GitHub

Images and Code

Requires: OSX, crontab


Inspired by a post on /r/DoctorWho I made a small applescript that gets the hour from the current time and sets the wallpaper to match to the correct Doctor. So if its 10pm or 10am the David Tennant wallpaper will be shown for the hour. I then used crontab to run this script every minute so that it would switch on the hour.

I choose applescript because it integrated well with OSX, much better than any other portable language. My first attempt at this was doing it in python but that required lots of strange imports and stranger syntax to get python to talk to the operating system to change the desktop background. With applescript it was an easy "tell finder to set desktop background" type statement.

How too Install

If you have a mac and are a fan of DoctorWho and want this, heres how to do it.

  • Download the images here
  • Unzip the file and put it where ever you'd like
  • Open up the "DoctorClock.scpt" and change

    /Users/pgicking/pgicking@gmail.com/DoctorClock/

    to where you put the images
  • Open up a terminal and type crontab -e and put:
    * * * * * osascript "path to your DoctorClock.scpt"

  • Save and exit the crontab session and enjoy!
